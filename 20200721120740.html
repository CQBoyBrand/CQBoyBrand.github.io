2020-今年的裸辞暂告一个段落 | 我的年中总结

发表于 2020-07-21 阅读次数：

5月20日从上家公司拿到离职证明，到7月20日接了一个自认为还算满意的 offer，这个为期2个月、从 杭州到成都 的 裸辞 之旅暂告一个段落。

5月20日从上家公司拿到离职证明，到7月20日接了一个自认为还算满意的 offer，这个为期2个月、从杭州到成都的裸辞之旅暂告一个段落。

本文主要分三个话题：

  1. [自我介绍]
  2. [最后总结]
  3. [一些面试题]

自我介绍

2016年某大学电子信息科学与技术专业毕业，自学踏入了前端这个行业，到2020年的7月，算是整整4年的前端经验了，一直混迹在各种不知名的小厂，基本是自己一个人在前端这块单打独斗，本来是加一些「技术群」去向大佬取经的，结果都变成了闲扯吹水的群。自己也一天天从前端小白，变成了前端老白。

前两年 JQuery 一把梭，后两年 Vue 全家桶熟练工，新工作大部分时间要搞 React Hooks了。

工作第二年开始了解到 Vue，由于工作中没有用武之地，只好自己折腾，于是 2017 年自己开始用 Vue + Koa2 + Mysql 折腾上线了自己的博客：

  • http://www.brandhuang.com

18年准备换工作的时候，群里已经有朋友开始用 React 了，于是自己又开始学习 React ，想着有人一起学习，有问题方便找人请教，朋友推荐了《React小书》，照着撸了一遍了，又开始拿自己博客开始练手，尝试写写 demo：

  • https://github.com/CQBoyBrand/react-blog
  • https://github.com/CQBoyBrand/react-paging-component

谁料到，18年换了这份工作开始，至今，经历的公司都是用的 Vue。React 就这样被我放弃了，18年可以说是我 React 技术最牛逼的时候！

最近一年开始接触：数据结构与算法、设计模式、性能优化等等…

由于在工作中都没有机会大量的实战（或者说没有大佬带，自己一个人不敢大量实战），所以大多数都是浅尝辄止，学习也是属于两天打鱼，三天晒网，还有两天在想到底要不要学这个，学了到底有不有用。

我这四年经验换算成其他同行的，可能就3个月的经验吧

我发现 「工作中没有用到」 成了我学习技术路上最大的绊脚石

新工作，新开始，立个 Flag：从现在开始打磨这块绊脚石，争取早日碾碎它

这次裸辞虽然内心慌的一批，但是表面还是要装出一副很镇定的样子，毕竟是男人，决不允许让别人说自己不行，即使真不行，也能让别人知道！

今年的面试除了和项目强相关的外，其他基础题基本都能在掘金别的大佬的面经上刷到

自己今年的面试总结起来大概就是：「js基础 30%」、「框架原理 30%」、「简历上写的内容 30%」、「性能优化 4%」、「计算机网络 4%」、「数据结构与算法 1%」、「其他 1%」

总结

不知道自己这篇文章到底要表达个什么意思，就当作自己的一个年中总结吧。

今年概况：

  • 2020年2月份复工，上家公司薪资微调，不符预期
  • 3月底公司调整，换项目组，
  • 4月下旬提离职裸辞
  • 5月底离开杭州回到成都
  • 7月下旬入职新公司，薪资涨幅还算满意（比上家公司好吧）
  • 下半年新环境，事太多，好好学习，奥利给！

以个人这四年混过来的经验来说，为了混口好饭吃，这几点真的很重要：

  • 制定目标：方向错了，可能怎么做都是错的；
  • 做好计划：规划时间，摆脱拖延症；画好图（脑图、UML类图、流程图等等），按部就班；
  • 自学能力：找到适合自己的学习方式和内容；
  • 动手实战：实践出真知，是骡子是马拉出来遛遛；
  • 做好记录：好记性不如烂笔头，随时翻看，减缓记忆衰减；
  • 持之以恒：三分钟热度最终都会凉凉，除了浪费时间什么也得不到

学习的路还很长，虽然苦，但值得咬牙坚持。

每天有一点收获也是一种进步

我相信和我一样 间歇性踌躇满志，持续性混吃等死 的人应该不在少数

只希望你回顾过去的时候，不会后悔
